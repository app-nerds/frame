Cards

Card Demonstration

This is the first card. It contains content. The markup is simple.

First: Container

Start with a DIV that has a class of .cards.

<div class="cards">

</div>

Second: Sections

Then add sections that look like this:

<section>
  <h3>Title Goes Here</h3>

  <p>
    Content goes in here. Put what you want!
  </p>

  <footer>
    <button>Button</button>
  </footer>
</section>

All Together!

That's it! Here's a full sample!

<div class="cards">
  <section>
    <h3>Title Goes Here</h3>

    </p>

    <footer>
      <button>Button</button>
    </footer>
  </section>
</div>
